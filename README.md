# Movie Explorer React

Este é um projeto desenvolvido em React cujo objetivo é permitir aos usuários explorar e descobrir informações sobre filmes. O **Movie Explorer React** consome uma API externa de filmes para apresentar listas, detalhes, trailers e outras informações relevantes do mundo do cinema.

**🔗 Acesso Online (Vercel):** [https://movie-explorer-react-five.vercel.app/](https://movie-explorer-react-five.vercel.app/)

## 🚀 Tecnologias e Ferramentas

O projeto utiliza o ecossistema moderno do React, incluindo as seguintes ferramentas:

- **React 19:** Biblioteca principal para a criação de interfaces de usuário.
- **Vite:** Ferramenta de build extremamente rápida que oferece uma excelente experiência de desenvolvimento com HMR.
- **React Router DOM 7:** Utilizado para gerenciar a navegação e as rotas dentro da SPA (Single Page Application).
- **Axios:** Cliente HTTP baseado em Promises utilizado para realizar requisições à API de filmes.
- **CSS Modules / SCSS (opcional/conforme estilo global do projeto):** Aplicados para estilos independentes e de escopo restrito aos componentes.

## ⚙️ Funcionalidades

- Listagem de filmes.
- Exibição de detalhes de um filme específico (sinopse, avaliação, etc).
- Trailer e informações gerais.
- Navegação entre rotas de forma ágil e fluída.

## 🛠️ Como Executar o Projeto Localmente

1. **Clone este repositório:**
   ```bash
   git clone https://github.com/DyegoMelo20/MovieExplorerReact.git
   ```

2. **Acesse a pasta do projeto:**
   ```bash
   cd MovieExplorerReact
   ```

3. **Instale as dependências:**
   ```bash
   npm install
   ```

4. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

5. **Abra o seu navegador e acesse:**
   O terminal do Vite indicará a URL padrão, geralmente `http://localhost:5173`.

## 📦 Deploy

O projeto também está configurado para publicação no **GitHub Pages**. Para gerar uma build de produção e enviar automaticamente:

```bash
npm run deploy
```
